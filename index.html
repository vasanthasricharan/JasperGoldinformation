Functional Property Verification (FPV)

JasperGold FPV is the industry's highest-capacity formal solver. it proves SystemVerilog Assertions (SVA) across all design states to catch deep corner-case bugs.

// SVA Verification Logic
property req_gnt_check;
  @(posedge clk) req |-> ##[1:4] gnt;
endproperty
assert_check: assert property(req_gnt_check);

Connectivity Verification

Automate the verification of SoC-level wiring. It structurally proves signal paths, identifies inversions, and detects floating pins in massive netlists.

// Connectivity Command
check_conn -from {TOP.cpu_out} -to {TOP.mem_in} -expected 1

Property Synthesis

Automatically generates SystemVerilog Assertions (SVA) by observing RTL behavior. This jumpstarts verification for legacy code where no assertions exist.

Low Power (UPF) Verification

Verifies designs using Unified Power Format (UPF). It formally proves that isolation cells and level shifters operate correctly during power-down/up sequences.

Power Domain Analysis

Security Path Verification

Ensures hardware security by performing formal Taint Analysis. It proves that sensitive data (like keys) never leaks to unauthorized non-secure ports.

Security Vulnerability Check

Sequential Logic Equivalence (SLEC)

Proves that two designs are functionally identical over time. Used to verify RTL vs. optimized netlists or hardware acceleration blocks.
